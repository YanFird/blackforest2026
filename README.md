# Black Forest 2026 Family Trip Website

Public static website for the Black Forest + Alsace family trip brochure.

## Live site

After GitHub Pages is enabled, the site should be available at:

https://yanfird.github.io/blackforest2026/

## Files

- `index.html` — the Hebrew interactive travel brochure.
- `assets/` — local images used by the brochure.
- `.nojekyll` — tells GitHub Pages to serve the static files directly.

## Update workflow

From this folder:

```bash
git status
git add .
git commit -m "Update trip website"
git push
```
